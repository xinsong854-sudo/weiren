# 伪人大本营 - 互动视觉小说

## 世界观设定
- **伪人大本营** = 论坛（伪人交流平台）
- **烛灯** = 揭秘人组织最后成员，留下 4 篇生存指南帖子
- **营长** = 论坛管理员 + 槐安公寓楼长
- **伪人** = 模仿人类的生物（眨眼规律、重复话语、无情感）

## 剧情大纲

### 第一章：异常
**场景**：办公室
**剧情**：主角发现同事张明行为异常
**检定**：INS 检定（观察张明）
- 成功：发现张明是伪人
- 失败：不确定，继续观察
- 大成功：发现更多细节
- 大失败：被张明发现你在观察他

### 第二章：真相
**场景**：办公室/回家路上
**剧情**：发现烛灯的帖子，了解伪人真相
**检定**：SAN 检定（面对真相）
- 成功：接受现实
- 失败：心理受创
- 大成功：冷静分析
- 大失败：精神崩溃

### 第三章：选择
**场景**：家中/公寓
**剧情**：决定如何应对
**检定**：DIS 检定（伪装自己）
- 成功：融入人群
- 失败：引起怀疑
- 大成功：完美伪装
- 大失败：身份暴露

### 第四章：生存
**场景**：城市/槐安公寓
**剧情**：寻找生存方法
**检定**：LUK 检定（随机事件）
- 成功：遇到帮助
- 失败：遭遇危险
- 大成功：意外收获
- 大失败：倒霉透顶

### 第五章：结局
**场景**：多种
**检定**：SUR 检定（最终生存）
根据之前检定结果和选择，导向不同结局

## 结局分支

### Bad Ending (6 个)
1. **被捕获** - 被伪人发现并捕获
2. **身份暴露** - DIS 大失败
3. **精神崩溃** - SAN 大失败
4. **倒霉透顶** - LUK 大失败
5. **孤立无援** - 错误选择
6. **成为同类** - 被转化

### Normal Ending (6 个)
1. **幸存者** - 基础存活
2. **逃离城市** - 成功逃脱
3. **隐藏身份** - 低调生活
4. **寻求庇护** - 加入猎人
5. **完美市民** - DIS 大成功
6. **重新开始** - 失忆重启

### True Ending (5 个)
1. **幕后操控者** - INS 大成功 + 正确选择
2. **揭秘人继承者** - 找到烛灯遗产
3. **论坛管理员** - 接触营长
4. **双重间谍** - 平衡双方
5. **真相揭露** - 公开真相

## 检定系统

### 属性
- **SAN (理智)**: 抵抗恐惧、保持冷静
- **INS (洞察)**: 发现异常、分析线索
- **DIS (伪装)**: 融入人群、扮演身份
- **LUK (幸运)**: 随机事件、意外发现
- **SUR (生存)**: 潜入、抵抗伤害

### 计算公式
```
成功线 = 属性值 + 技能值 - DC + 50
```

### 结果判定
- **大成功**: 骰子 1-5 → 专属好结局
- **成功**: 骰子 ≤ 成功线 → 通过
- **失败**: 骰子 > 成功线 → 未通过
- **大失败**: 骰子 96-100 → 专属坏结局

## 场景流程

```
start (开始)
  ↓
observe_colleague (观察同事) → INS 检定
  ↓
  ├─ 大成功 → discover_more (发现更多)
  ├─ 成功 → confirm_pseudo (确认伪人)
  ├─ 失败 → unsure (不确定)
  └─ 大失败 →被发现 → captured (被捕获) [BAD ENDING]
  
confirm_pseudo / discover_more / unsure
  ↓
find_forum (发现论坛)
  ↓
read_posts (阅读帖子) → SAN 检定
  ↓
  ├─ 大成功 → calm_analysis (冷静分析)
  ├─ 成功 → accept_truth (接受真相)
  ├─ 失败 → scared (恐惧)
  └─ 大失败 → breakdown (精神崩溃) [BAD ENDING]
  
calm_analysis / accept_truth / scared
  ↓
make_choice (做出选择)
  ↓
  ├─ 伪装自己 → disguise (伪装) → DIS 检定
  ├─ 逃离城市 → escape (逃离) → LUK 检定
  ├─ 寻求庇护 → seek_help (求助) → SUR 检定
  └─ 调查真相 → investigate (调查) → INS 检定
  
disguise (DIS 检定)
  ↓
  ├─ 大成功 → perfect_cover (完美伪装) [NORMAL ENDING]
  ├─ 成功 → blend_in (融入人群)
  ├─ 失败 → suspicious (引起怀疑)
  └─ 大失败 → exposed (身份暴露) [BAD ENDING]
  
escape (LUK 检定)
  ↓
  ├─ 大成功 → lucky_escape (幸运逃脱)
  ├─ 成功 → escape_success (成功逃离) [NORMAL ENDING]
  ├─ 失败 → blocked (被阻拦)
  └─ 大失败 → unlucky (倒霉透顶) [BAD ENDING]
  
investigate (INS 检定)
  ↓
  ├─ 大成功 → find_candlelight (找到烛灯遗产) [TRUE ENDING]
  ├─ 成功 → find_clues (发现线索)
  ├─ 失败 → lost (迷失)
  └─ 大失败 → trapped (陷入陷阱)
  
最终场景根据之前检定结果导向不同结局
```
